## 🧪 Low-Leakage Junctionless Vertical TFET Gas Sensor

**Tools Used**

- Silvaco TCAD
- ATLAS
- DeckBuild

### Description

Designed and simulated a Gate-Stack Junctionless Vertical Tunnel Field Effect Transistor (GSJLVTFET) for hydrogen and ammonia gas sensing applications. Investigated the impact of catalytic gate materials and work-function engineering on electrical characteristics and sensor sensitivity.

### Key Features

- TFET Device Simulation
- Hydrogen Gas Detection
- Ammonia Gas Detection
- Low Leakage Current
- Transfer Characteristics Analysis
- Sensitivity Analysis
- Electric Field Analysis
- Energy Band Analysis

### Technologies

Silvaco TCAD • Semiconductor Devices • Device Simulation

## 🚗 Vehicle Pollution Detection System

**Tools Used**

- Arduino UNO
- MQ-135 Gas Sensor
- Embedded C
- LCD Display

### Description

Designed and developed a real-time embedded system for monitoring vehicle exhaust emissions using the MQ-135 gas sensor. The system continuously measures harmful gas concentration, displays live readings on an LCD, and generates alerts when pollution exceeds predefined safety limits.

### Key Features

- Real-time gas monitoring
- Air quality measurement
- Threshold-based pollution alert
- LCD display integration
- Low-cost IoT monitoring solution

### Technologies

Arduino • Embedded C • MQ-135 • Sensors
